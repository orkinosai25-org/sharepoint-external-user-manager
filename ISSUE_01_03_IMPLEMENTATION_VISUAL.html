ISSUE 3: Plan Limits Enforcement

Status: Client space limits fully implemented and tested

Plan Configuration

Starter

• Max 5 Client Spaces
• Max 50 External Users
• $29/month

Professional

• Max 20 Client Spaces
• Max 250 External Users
• $99/month

Business

• Max 100 Client Spaces
• Max 1,000 External Users
• $299/month

Enterprise

• Unlimited Client Spaces
• Unlimited External Users
• $999/month

Enforcement Implementation

  • PlanEnforcementService - Centralized service for all limit checks
  • ClientsController Integration - Enforces limits before client creation
  • Error Response - Returns 403 Forbidden with PLAN_LIMIT_EXCEEDED error code
  • User-Friendly Messages - Includes current plan name and upgrade prompt
  • Feature Gating - Restricts features based on plan tier

Known Limitations

  • External User Limit Enforcement - Not yet implemented (requires database tracking)

Why: External users are managed in SharePoint, not in the local database. Fetching counts from SharePoint for every invitation would be expensive and unreliable. A proper solution requires implementing an ExternalUsers table that mirrors SharePoint data.
