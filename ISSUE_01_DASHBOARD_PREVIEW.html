Install SPFx Client

To use the ClientSpace web parts in your SharePoint tenant, install the SPFx package:

  1. Download the latest .sppkg package file
  2. Go to your SharePoint Admin Center → More features → Apps
  3. Open the App Catalog
  4. Upload the .sppkg file
  5. Check "Make this solution available to all sites in the organization"
  6. Click "Deploy"
